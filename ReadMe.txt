This is the 2nd prototype of the museum app.
The screens are in the self-explanatory files, MapScreen, ListScreen, RoomScreen, MusicScreen, and CreditScreen.
All the files are written in JavaScript, and the app is built with React Native.
Here's the link to installing react native:
https://facebook.github.io/react-native/docs/getting-started.html

After installing the react native, run these commands:
cd SimpleApp
react-native run-ios
